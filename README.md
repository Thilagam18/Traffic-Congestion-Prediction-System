# Traffic Congestion Prediction and Route Optimization Using Data Analytics

## Project Overview

This project helps monitor traffic conditions, predict congestion levels, and recommend optimized travel routes using data analytics techniques.

## Features

- User Login
- User Registration
- Traffic Data Management
- Traffic Analytics Dashboard
- Data Visualization Charts
- Traffic Congestion Prediction
- Machine Learning Dashboard
- Route Optimization
- User Management
- Report Generation
- Real-Time Monitoring

## Technologies Used

### Frontend

- React JS
- React Router DOM
- Axios

### Backend

- Spring Boot
- REST API

### Database

- MySQL
