# Traffic Congestion Prediction and Route Optimization System

## Project Overview
Traffic congestion is a major issue in urban areas, causing delays, increased fuel consumption, and environmental pollution. This project aims to predict traffic congestion levels using historical and real-time traffic data and suggest optimized routes to users.

## Objectives
- Predict traffic congestion levels.
- Suggest alternative routes.
- Reduce travel time.
- Improve traffic management efficiency.

## Technologies
- Frontend: React.js
- Backend: Spring Boot
- Database: MySQL
- Machine Learning: Python, Scikit-Learn
- Tools: GitHub, Postman, VS Code
